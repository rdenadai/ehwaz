# Ehwaz

Ehwaz is a native code editor built in Rust with Slint.
It focuses on speed, split-pane editing, strong keyboard workflows, and local-first tooling.

Current version: `0.0.3`

Hosted on: https://git.rdenadai.dev/rdenadai/ehwaz
